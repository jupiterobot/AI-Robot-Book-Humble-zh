# happy_parameter
## 概要
第2章的示例程序。参数通信。

## 安装
Chapter2的包需要全部一起安装和构建。
- 请参考[第2章 安装](https://github.com/AI-Robot-Book/chapter2)。

## 运行
- 将终端分为两个，在上方的终端中执行以下命令：
cd ~/airobot_ws
source install/setup.bash
ros2 run happy_parameter happy_parameter_server_node

- 在下方的终端中执行以下命令：
cd ~/airobot_ws
source install/setup.bash
ros2 run happy_parameter happy_parameter_client_node
