# waypoint_navi
## 概要
第4章的路点导航示例程序。从 ROS2 Humble 开始，可以使用名为 Nav2 Simple Commander 的 Python API，使得编写导航程序变得更加简单。本程序基于此 API，对初版（ROS2 Foxy 版本）的路点导航示例程序进行了全面重写。详细内容请参见第2版。

## 安装
第4章的软件包将统一进行安装与构建。
- 请参考 [第4章 安装](https://github.com/AI-Robot-Book/chapter4)。


## 实行
请事先启动真实机器人或模拟器。在终端中执行以下命令：
cd ~/airobot_ws
source install/setup.bash
ros2 run waypoint_navi waypoint_navi
