# 地图文件

## 概要
第4章用于导航的地图文件。  
该地图文件 house.pgm 和 house.yaml 是对参考文献内容进行修改后的版本。

## 安装
根据书中说明，map 目录的路径为 $HOME/map，因此请使用以下命令复制文件：
cp -r ~/airobot_ws/src/chapter4/map ~
