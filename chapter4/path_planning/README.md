# path_planning
## 概要
第4章 路径规划的示例代码。这不是一个 ROS2 软件包。

## 安装
第4章的软件包将统一进行安装与构建。
- 请参考 [第4章 安装](https://github.com/AI-Robot-Book/chapter4)。

## 实行
进入代码所在目录：
cd ~/airobot_ws/src/chapter4/path_planning
将终端分为三部分，分别执行以下命令：
python3 path_planning.py 1      # 广度优先搜索
python3 path_planning.py 2      # Dijkstra 算法
python3 path_planning.py 3      # A* 算法
