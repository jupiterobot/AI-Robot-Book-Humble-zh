# happy_teleop
## 概要
第4章的示例程序。自制版 teleop 控制器。

## 安装
第4章的软件包将统一进行安装与构建。
- 请参考 [第4章 安装](https://github.com/AI-Robot-Book/chapter4)。


## 实行
请事先启动模拟器。在终端中执行以下命令：
cd ~/airobot_ws
source install/setup.bash
ros2 run happy_teleop happy_teleop_node
